# China Labels Claude Code Tracker a Backdoor

## What Happened
China's national vulnerability database officially called Anthropic's Claude Code hidden telemetry a **backdoor**, one day after the original exposure. Both sides disagree on intent: China frames it as malicious surveillance, Anthropic frames it as abuse prevention.

## Why It Matters
- The same hidden-tracking mechanism is now a geopolitical incident.
- Official vulnerability-database classification escalates the dispute beyond technical criticism.
- Reinforces concerns about hidden model telemetry and how AI competition shapes product behaviour.

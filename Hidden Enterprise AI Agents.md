# Hidden Enterprise AI Agents

## What Happened
A CSA survey found **82% of enterprises have AI agents they do not know about**.

## Why It Matters
- Unregistered agents bypass policy and logging.
- Tool using agents can access data, send mail, and call APIs outside normal controls.
- Shadow agents become data exfiltration and prompt injection paths.

## Notes
- Use agent inventory and gateway controls where possible
- Treat every agent as a potential attack surface

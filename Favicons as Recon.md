# Favicons as Attack Surface Intelligence

## What Happened
A security write up showed how **browser favicons** can be used for passive reconnaissance and software fingerprinting.

## Why It Matters
- Tiny, easy to miss signals still expose software stacks.
- Attackers can map honeypots, services, and environments at scale.
- Defence teams often overlook static asset fingerprinting.

## Notes
- Include assets beyond main HTML in scoping
- Treat public icons, headers, and metadata as disclosure

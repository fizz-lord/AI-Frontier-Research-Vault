# Pliny T3MP3ST Attack Harness

## What It Is
Pliny released **T3MP3ST**, an autonomous red teaming platform described as a multi agent offensive security meta harness.

## Why It Matters
- Offensive tooling is becoming reusable, shareable, and repeatable.
- Attack pipelines can target multiple apps, APIs, or agents in one run.
- Defenders need tests that assume AI assisted adversaries.

## Key Points
- Autonomous multi agent offensive workflow
- Focused on model, application, and API attack surface
- A signal that red teaming tooling itself is commoditising
